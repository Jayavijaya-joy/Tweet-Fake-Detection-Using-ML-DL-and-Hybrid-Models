About TweepFake Detection

TweepFake Detection is a Machine Learning based system developed to identify whether a tweet is written by a real human user or generated by automated bot accounts. With the rapid growth of social media platforms, bot-generated content has increased significantly, affecting public opinion, marketing authenticity, and digital trust.

This project uses Natural Language Processing (NLP) techniques combined with supervised Machine Learning algorithms to analyze linguistic patterns, writing styles, and text structure of tweets.

🎯 Project Objectives

  • Detect fake bot-generated tweets
  • Improve social media authenticity analysis
  • Provide real-time tweet classification
  • Demonstrate practical NLP + ML implementation

⚙️ Technology Stack

  • Frontend: HTML, CSS, JavaScript
  • Backend: Python Flask API
  • ML Models: TF-IDF + ML Classifiers
  • Deployment: Render + Netlify

🧠 Core ML Workflow

Text → Cleaning → Tokenization → TF-IDF Vectorization → Model Prediction → Result Visualization

Dataset Statistics & Distribution

The TweepFake dataset contains approximately 2558 tweet samples collected from real human users and automated bot accounts. The dataset is used to train supervised machine learning models.

📊 Dataset Columns

  • screen_name → Twitter username
  • text → Tweet content
  • account.type → Bot or Human account category
  • class_type → Final classification label

📈 Dataset Insights

  • Balanced distribution between Human and Bot tweets
  • Contains real-world social media language patterns
  • Includes informal text, emojis, and abbreviations
  • Useful for NLP classification benchmarking

Machine Learning Models Used

Multiple machine learning approaches can be used for tweet classification. Below are the core model architectures supported in this project.

TF-IDF + Logistic Regression

Converts text into numerical vectors using Term Frequency – Inverse Document Frequency. Logistic Regression then classifies tweet probability into Bot or Human class.

✔ Fast Training ✔ High Interpretability ✔ Good Baseline Performance

Recurrent Neural Networks (RNN)

Captures sequential dependencies in tweet text. Useful for understanding writing flow and sentence structure patterns.

✔ Sequence Learning ✔ Context Awareness ✔ Deep NLP Capability

Transformer Models

Uses attention mechanism to understand deep contextual relationships between words. Provides state-of-the-art NLP performance.

✔ Highest Accuracy ✔ Contextual Understanding ✔ Industry Standard NLP
