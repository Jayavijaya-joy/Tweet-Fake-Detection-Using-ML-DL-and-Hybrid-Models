# AI-Generated Tweet Detection System

## Project Overview
This project focuses on detecting AI-generated (deepfake) tweets using advanced Machine Learning and Deep Learning techniques. The system helps identify misinformation on social media by analyzing tweet content and classifying it as genuine or fake.

---

## Features
- Detects AI-generated tweets with high accuracy
- Uses hybrid model (FastText + CNN-LSTM + BERT)
- Data preprocessing and feature engineering
- Real-time tweet analysis capability
- Visualization of results (graphs & heatmaps)
- Improved model performance with hyperparameter tuning

---
##  Technologies Used
- Python
- Machine Learning (Scikit-learn)
- Deep Learning (TensorFlow / Keras / PyTorch)
- FastText Embeddings
- BERT Model
- Pandas, NumPy
- Matplotlib, Seaborn

---

##  Dataset
- Dataset contains ~19,000 tweets
- Cleaned and preprocessed dataset used
- Includes both real and AI-generated tweets

---

##  Methodology
1. Data Collection
2. Data Preprocessing
3. Feature Engineering (TF-IDF, FastText)
4. Model Building:
   - CNN-LSTM
   - BERT
5. Hybrid Model Integration
6. Model Evaluation (Accuracy, F1 Score)

---

## Results
- Achieved high accuracy in detecting AI-generated tweets
- Hybrid model outperformed individual models
- Improved classification performance using ensemble approach
